# SKILL: train_model

Kick off offline training jobs for Hannah's backend models from the CLI.

Supported model ids:
- `tyre_model`
- `laptime_model`
- `pit_rl`
- `pit_policy_q`
- `winner_ensemble`
- `all`

This tool is for explicit retraining only. Do not use it to answer one-off race strategy questions.
